# Purabo Recipe Store 📖

The **Purabo Recipe Store** is a community-driven repository of optimized configurations for web-to-desktop apps.

## Current Recipes

### 1. WhatsApp
- **URL:** `https://web.whatsapp.com`
- **Icon:** Official Logo
- **Tweak:** Fixed aspect ratio for messaging.

### 2. Claude
- **URL:** `https://claude.ai`
- **Icon:** Anthropic/Claude Logo
- **Tweak:** Minimalist header injection.

### 3. Notion
- **URL:** `https://notion.so`
- **Icon:** Official Logo
- **Tweak:** Faster loading flags.

## Roadmap
In the next phase, Purabo will fetch these recipes from a remote JSON hosted on GitHub. This will allow:
- Instant updates without app restarts.
- Community PRs for new "Optimized" recipes.
- Themed icons for different desktop environments.
